# Termux-Systemctl  (shell)
run [systemctl3.py](https://github.com/gdraheim/docker-systemctl-replacement) in [termux](https://github.com/termux/termux-app)  

# Requirements
* Python3                       (necessary)
* sudo (in proot/chroot Linux)  (optional)

# Systemctl applet
journalctl) `journalctl`  
init)       `systemctl init`  
halt)       `systemctl halt`  
reboot)     `init&&halt`  
*)          `systemctl`
